# GPX编辑器 - 项目进度文档

> 最后更新: 2026-05-19 (航迹工具 + 地图框选编辑)

## 一、项目概况

- **项目名称**: GPX编辑器 v1.0.0
- **定位**: GPX航点航迹编辑处理工具，支持GUI界面操作，提供航点/航迹基础操作和TXT/GDB格式导出功能
- **技术栈**: Python 3.10+ / tkinter + ttkbootstrap / gpxpy / tkintermapview / Pillow
- **入口文件**: `main.py`，启动 `MainWindow` 的 tkinter 主循环
- **启动脚本**: `GPX编辑器.bat`，使用 `pythonw main.py` 无控制台启动

## 二、模块结构

```
gpx_editor/
├── core/                        # 核心处理层
│   ├── gpx_handler.py           # GPX文件读写
│   ├── waypoint.py              # 航点数据模型与操作
│   ├── track.py                 # 航迹数据模型与操作
│   ├── track_stats.py           # 航迹统计（距离/速度/方向/海拔剖面）
│   ├── exporter.py              # TXT/GDB格式导出
│   ├── gpx_editor.py            # GPX属性编辑（文本级正则操作）
│   ├── batch_matcher.py         # 批量航点匹配
│   ├── coord_converter.py       # 坐标系转换
│   ├── excel_exporter.py        # Excel导出
│   └── verifier.py              # 三方数据一致性校验
├── gui/                         # GUI界面层
│   ├── main_window.py           # 主窗口（含航迹点显示/框选/拖动）
│   ├── waypoint_dialog.py       # 航点编辑对话框
│   ├── track_dialog.py          # 航迹编辑对话框
│   ├── track_operations_dialog.py # 航迹操作对话框（简化/合并/分割）
│   ├── undo_manager.py          # 撤销/重做管理器
│   ├── properties_dialog.py     # 属性对话框（MapSource风格分标签页，含航迹属性）
│   ├── elevation_chart.py       # 海拔剖面图组件（Canvas绘制）
│   ├── move_waypoint_dialog.py  # 移动航点对话框（地图点击/手动输入）
│   ├── gpx_editor_dialog.py     # GPX属性编辑器对话框
│   ├── batch_export_dialog.py   # 批量导出对话框
│   ├── batch_match_dialog.py    # 批量航点匹配对话框
│   ├── verification_dialog.py   # 三方数据校验对话框
│   └── column_config_dialog.py  # 列配置对话框
└── utils/
    └── helpers.py               # 通用工具函数
```

## 五、当前可用功能

程序启动后，**主窗口**已实现完整的航点航迹编辑功能：

### 主窗口功能
1. **文件操作** -- 新建/打开/保存/另存为GPX文件
2. **航点编辑** -- 添加/编辑/删除航点，支持对话框输入坐标
3. **航迹编辑** -- 添加/编辑/删除航迹
4. **树形列表** -- 分组显示航点和航迹，支持选择、双击编辑、右键菜单
5. **地图显示** -- 高德地图路网标注，可切换卫星图层
6. **地图交互** -- 鼠标悬停显示坐标，右键菜单"在此添加航点"，自动缩放到数据范围
7. **撤销/重做** -- Ctrl+Z/Ctrl+Y支持
8. **快捷键** -- Ctrl+N/O/S、Delete、Ctrl+C/V/X、Ctrl+Shift+W/T
9. **航点右键菜单** -- 在地图显示、航点属性、复制/粘贴/剪切、移动航点、删除
10. **航迹右键菜单** -- 在地图显示、航迹属性、复制/粘贴/剪切、航迹操作、删除
11. **属性对话框** -- MapSource风格分标签页（基本信息/位置坐标/详细描述）
12. **航迹点编辑** -- 地图上显示绿色小圆点，支持框选、拖动移动、右键菜单
13. **航迹属性对话框** -- 三标签页（航迹点列表/统计汇总/海拔剖面图），与地图选中同步
14. **比例尺** -- 地图右下角实时线段+距离显示
15. **地图瓦片缓存** -- SQLite磁盘缓存，加速二次加载
16. **多图层** -- 高德路网/天地图卫星/OSM，缩放级别扩展至19-20级

### 工具菜单功能
1. **GPX属性编辑器** -- 解析/修改GPX属性、坐标偏移（单文件+批量）
2. **批量导出** -- GPX转TXT/GDB格式（单文件+批量目录）
3. **批量匹配航点** -- 基线匹配、删除未匹配航点、重命名（含预览）
4. **航迹操作** -- 简化（Douglas-Peucker）、合并、分割航迹
5. **三方数据校验** -- 对比分配航点、GPS实际航点、样品编号（单次+批量，支持Excel导出）
