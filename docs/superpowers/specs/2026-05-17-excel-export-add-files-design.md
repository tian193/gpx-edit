# Excel导出对话框添加文件功能设计

## 背景

Excel导出对话框中已有 `_add_files()` 和 `_add_folder()` 方法，但未连接到任何UI按钮，导致用户只能导出当前打开的GPX文件中的航点，无法添加其他GPX文件。

## 设计目标

在Excel导出对话框中添加"添加文件"和"添加文件夹"按钮，让用户可以在导出时选择多个GPX文件。

## 改动范围

仅修改 `gpx_editor/gui/excel_export_dialog.py` 的 `_create_widgets()` 方法。

## 具体改动

在"选择航点"面板的按钮栏中，在"全选"按钮左侧添加两个按钮：

**当前布局：**
```
[全选] [取消全选]
```

**改动后：**
```
[添加文件] [添加文件夹] [全选] [取消全选]
```

### 按钮绑定

- "添加文件" → `_add_files()`（已实现，支持多选文件）
- "添加文件夹" → `_add_folder()`（已实现，扫描目录下所有.gpx文件）

### 无需改动

- `_add_files()` 和 `_add_folder()` 方法已完整实现
- 自动加载当前文件的行为保持不变
- 无架构变更
