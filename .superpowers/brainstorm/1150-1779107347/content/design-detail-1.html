CGCS2000投影坐标计算

实现方式

  • 使用proj库或纯Python实现WGS84→CGCS2000转换
  • 6度带，中央经线 = 带号 × 6 - 3
  • 带号 = floor(第一个航点经度 / 6) + 1
  • 计算列在数据加载时批量计算，缓存结果
  • 列头显示带号信息，如"CGCS2000 X (39带)"

依赖

  • 需要安装pyproj库（推荐）或自行实现坐标转换
  • pyproj基于PROJ库，精度高，支持多种投影

配置持久化

配置文件格式 (column_config.json)
{
  "visible_columns": ["type", "name", "lat", "lon", "ele", "time"],
  "column_order": ["type", "name", "lat", "lon", "ele", "time", "desc", "cmt", "sym", "source", "cgcs2000_x", "cgcs2000_y"],
  "column_widths": {
    "type": 60,
    "name": 120,
    "lat": 100,
    "lon": 100,
    "ele": 80,
    "time": 140
  },
  "cgcs2000_zone": null
}
